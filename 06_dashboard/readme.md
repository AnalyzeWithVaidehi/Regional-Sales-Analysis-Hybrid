# 📊 Strategic Sales Dashboard (v1.0)

This folder contains the Power BI visualizations for the **2026 Regional Sales Forecast**. This dashboard translates the Seasonal Naïve model results into actionable business insights.

## 🔍 Version 1.0 Highlights
*   **2026 Revenue Forecast:** Projected total revenue of **$15.79M**.
*   **Gap Analysis:** Real-time tracking against the **$17.37M** budget, identifying a current **-$1.58M (-9.09%) revenue gap**.
*   **Trend Visualizations:** Comparative line charts showing Actual vs. Budget vs. Forecasted revenue from 2023 through 2026.
*   **Monthly Breakdown:** Detailed monthly table and bar charts identifying October 2026 as the month with the highest forecasted revenue (~$1.5M).
